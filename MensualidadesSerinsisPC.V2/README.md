# MensualidadesSerinsisPC.V2

Solución nueva e independiente para clientes, suscripciones, bases de datos, facturación, pagos, cartera, automatización e ingresos mensuales.

## Proyectos

- `MensualidadesSerinsisPC.V2.Web`
- `MensualidadesSerinsisPC.V2.Domain`
- `MensualidadesSerinsisPC.V2.Application`
- `MensualidadesSerinsisPC.V2.Infrastructure`
- `MensualidadesSerinsisPC.V2.Worker`

## Base de datos

Script inicial en:

- `Database/DBMensualidadesSerinsisPC_V2_Initial.sql`

Incluye:

- clientes
- planes
- suscripciones
- bases de datos por cliente
- facturas
- pagos y aplicaciones
- programacion de cobros
- notificaciones
- bitacoras
- vistas de ingresos, cartera y lectura para POS
- procedimientos para pago, programacion y actualizacion de estado
